「Reflow 온도 Profile 작성 기준」

## 📘 1. 목적

Reflow 공정에서 **PCB 온도 측정 및 표준 프로파일 설정 방법**을 규정하여

* 납땜 불량(Cold Solder, Bridge, Solder Ball 등)을 예방하고
* Reflow Oven의 온도 관리 정확도를 확보하는 것.

---

## ⚙️ 2. 온도 측정 방법

### (1) 사용 도구

| 구분                    | 설명                                         |
| --------------------- | ------------------------------------------ |
| **온도기록계(Profiler)**   | 열전대(Thermocouple)로부터 전압 신호를 받아 온도로 환산하는 장치 |
| ① Mole Type(무선)       | 무선 송수신 방식, PC·프린터 출력 가능, 작업 효율 우수          |
| ② Pen Type(유선)        | 바로 표시되며 즉시 확인 가능하나 작업성 낮음                  |
| **열전대(Thermocouple)** | 두 금속 접점 온도차에 따른 기전력을 이용해 온도를 측정            |

**열전대 종류 및 특성**

| Type | 재질                      | 일반측정온도 | 최대온도  | 수명(최대온도 기준) |
| ---- | ----------------------- | ------ | ----- | ----------- |
| B    | 백금로듐(30%) / 백금로듐(6%)    | 1500℃↓ | 1700℃ | 50시간        |
| R    | 백금로듐(13%) / 백금          | 1400℃↓ | 1600℃ | 50시간        |
| S    | 백금로듐(10%) / 백금          | 1400℃↓ | 1600℃ | 50시간        |
| N    | Ni14.2Cr1.4Si / Ni4.4Si | 1200℃↓ | 1250℃ | 250시간       |
| K    | Ni10Cr / Ni5AlMn        | 1000℃↓ | 1200℃ | 250시간       |
| E    | Ni10Cr / Cu55Ni45       | 700℃↓  | 800℃  | 250시간       |
| J    | Fe / Cu55Ni45           | 600℃↓  | 750℃  | 250시간       |
| T    | Cu / Cu55Ni45           | 300℃↓  | 350℃  | 250시간       |

---

### (2) 접속 방법

* **가장 정확한 방법:** +, – 피복을 벗기고 용접.
* **대안:** 꼬아서(약 10회) 절단 후 끝부분 약 3회 꼬임만 남기기.

**대표적 접속 방식**

| 방법            | 설명                                |
| ------------- | --------------------------------- |
| **고온 솔더법**    | 금속 리드 사이에 납땜 (Pb70/Sn30 이상 융점 사용) |
| **내열 테이프법**   | PCB 표면 등 비금속 부위 측정 시 간편           |
| **열경화 Bond법** | PCB 표면이나 부품 리드 온도 측정 시 고정용        |

**고온 솔더 종류:** Pb70/Sn30(255℃), Pb80/Sn20(280℃), Pb90/Sn10(300℃)

---

### (3) 측정 부위 선정

| Thermocouple 채널 수 | 측정 위치                                             |
| ----------------- | ------------------------------------------------- |
| **3채널 이하**        | ① 대형 부품 리드(QFP, BGA)<br>② 소형 부품(Chip)<br>③ 열취약 부품 |
| **6채널 이상**        | ①~② 동일 + 상/하면 비교용 채널<br>⑤~⑥ 고온취약 부품               |

---

### (4) 측정 절차

1. 금속부는 고온 솔더, 비금속부는 접착제 사용.
2. 내열테이프로 고정, 각 채널에 번호 라벨 부착.
3. 샘플 PCB를 Reflow 컨베이어에 투입하여 측정.
4. 결과를 표준 Profile과 비교하여 필요 시 Zone별 온도 조정.

---

### (5) 측정 주기

* 1일 1회 이상 또는 **모델 변경 시** 재측정
* 각 모델별 Profile 게시 및 관리
* Thermocouple 부착 PCB를 모델별로 보관

---

## 🌡️ 3. 온도 Profile 설정 기준

### (1) 표준 Profile (Sn63/Pb37, 용융 183℃ 기준)

| 항목                  | 표준값            | 실제 허용범위  |
| ------------------- | -------------- | -------- |
| **Peak 온도(A)**      | 225℃           | 200~240℃ |
| **200℃ 이상 유지시간(B)** | 25초            | 10~40초   |
| **예열 온도(C)**        | 150℃ ±5℃ / 75초 | 60~90초   |
| **승온 속도(D)**        | 2.5℃/초         | 3±0.5℃/초 |

**참고:**

* 200℃ 이상 유지시간은 납 완전 용융 확보에 필수.
* 과도한 예열·고온은 부품 열파괴 및 산화, 솔더볼 발생 원인.

---

## ⚠️ 4. 불량 유형별 온도 Profile 대책

| 불량유형                      | 원인(온도 프로파일 관점)     | 대책                 |
| ------------------------- | ------------------ | ------------------ |
| **냉납 (Cold Solder)**      | Peak 온도 과저온, 예열 과다 | 예열 온도 ↓, Peak 온도 ↑ |
| **납 브릿지 (Solder Bridge)** | 용융 점도 높음, 예열 온도 과고 | 예열 약간 ↓, Peak 온도 ↑ |
| **솔더볼 (Solder Ball)**     | 예열 부족 → 용제 미증발     | 예열 시간 연장           |
| **맨하탄 (Manhattan)**       | 예열-리플로우 온도차 과대     | 온도차 축소, 프로파일 재조정   |

---

## 📋 5. 관리 기준 요약

| 구분         | 관리 항목           | 주기    | 비고       |
| ---------- | --------------- | ----- | -------- |
| 온도 프로파일 측정 | 각 모델별           | 1회/일  | 변경 시 재측정 |
| Profile 게시 | Oven 전면         | 매일    | 최신값 유지   |
| 샘플 PCB     | Thermocouple 부착 | 상시 보관 | 모델별 관리   |
